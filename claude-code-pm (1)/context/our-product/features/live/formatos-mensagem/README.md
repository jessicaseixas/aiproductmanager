# Suporte a Multiplos Formatos de Mensagem

> O assistente compreende diferentes formas de comunicacao dos leads.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Leads se comunicam de formas variadas no WhatsApp (audio, emojis, reacoes). Um assistente que so entende texto limita a experiencia natural de conversa.

## Solução

Formatos suportados:
- Texto escrito
- Mensagens de voz (transcritas automaticamente)
- Emojis (ex: thumbs up para confirmar)
- Reacoes a mensagens

Resposta sempre em texto para consistencia.

## Usuário

Leads que preferem enviar audios ou usar emojis na comunicacao.

## Valor

- Experiencia mais natural e acessivel
- Maior taxa de resposta dos leads
- Inclusao de usuários que preferem audio a texto

## Dependências

Nenhuma.
