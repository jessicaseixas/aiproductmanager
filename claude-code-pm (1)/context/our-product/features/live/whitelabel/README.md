# Personalizacao Whitelabel

> Cada imobiliária opera o assistente em seu proprio número de WhatsApp com identidade visual propria.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Imobiliarias precisam que o atendimento automatizado represente sua marca, não uma solução generica. Leads devem sentir que estao falando com a imobiliária.

## Solução

Elementos personalizaveis por imobiliária:
- Foto do perfil do WhatsApp
- Nome exibido (ex: "Mais Imoveis", "Imobiliaria Negocio Certo")
- Descrição (seção "sobre") da conta
- Numero proprio de WhatsApp

## Usuário

Imobiliarias que querem manter sua identidade de marca no atendimento automatizado.

## Valor

- Experiencia consistente com a marca da imobiliária
- Maior confianca do lead no atendimento
- Flexibilidade para cada cliente operar com sua identidade

## Dependências

Nenhuma.
