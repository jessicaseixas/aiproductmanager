# Anota Duvidas e Dados Adicionais

> Quando o assistente não sabe responder uma pergunta, ele registra a duvida para repasse posterior ao corretor.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

O assistente não consegue responder todas as perguntas (ex: perguntas sobre a imobiliária, negociacao). Sem registro, essas duvidas se perdem e o corretor não sabe o que o lead quer.

## Solução

- Informa ao lead que a duvida sera respondida pelo corretor
- Registra a pergunta para repasse posterior
- Continua a conversa normalmente
- Captura informacoes adicionais fora do roteiro (ex: "Quero regiao silenciosa")

## Usuário

Leads com duvidas especificas; corretores que recebem o lead qualificado.

## Valor

- Nenhuma informacao do lead se perde
- Corretor recebe contexto completo do que o lead precisa
- Lead sente que foi ouvido mesmo sem resposta imediata

## Dependências

Tira Duvidas sobre Imoveis (complementa quando não há resposta).
