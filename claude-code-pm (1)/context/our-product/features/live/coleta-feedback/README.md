# Coleta de Feedback

> Ao final dos atendimentos, solicita avaliacao da experiencia.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Sem feedback dos leads, não há como medir a qualidade do atendimento automatizado nem identificar pontos de melhoria.

## Solução

- Solicita avaliacao ao final de cada atendimento
- Escala de 1-5 (Muito insatisfeito a Muito satisfeito)
- Campo opcional para comentarios (ate 600 caracteres)
- Registro para analise posterior

## Usuário

Gestores de produto e operacoes que monitoram qualidade do atendimento.

## Valor

- Metricas de satisfacao do atendimento (NPS/CSAT)
- Identificacao de problemas recorrentes
- Dados para melhoria continua do assistente

## Dependências

Nenhuma.
