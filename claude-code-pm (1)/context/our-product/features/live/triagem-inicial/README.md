# Triagem Inicial

> Identifica e separa leads de baixa prioridade, permitindo que corretores foquem nos contatos com maior chance de conversão.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Corretores perdem tempo significativo tentando contatar leads que não tem real interesse em comprar ou alugar. Leads que não respondem ou demonstram desinteresse consomem recursos sem retorno.

## Solução

- Filtra leads que não respondem as mensagens iniciais
- Identifica leads com desinteresse explicito ("era so curiosidade", "ja comprei", etc.)
- Categoriza automaticamente leads por nivel de prioridade
- Libera corretores para focarem em oportunidades reais

## Usuário

Corretores e gestores de vendas que recebem alto volume de leads.

## Valor

- Reducao de tempo gasto com leads improdutivos
- Aumento da taxa de conversão por lead trabalhado
- Maior satisfacao dos corretores com a qualidade dos leads

## Dependências

Nenhuma - feature core do produto.
