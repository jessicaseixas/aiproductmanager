# Tira Duvidas sobre Imoveis

> Responde perguntas dos leads utilizando informacoes cadastradas nos anuncios.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Leads fazem perguntas basicas sobre imóveis (vagas, condomínio, metragem) que poderiam ser respondidas automaticamente, mas acabam esperando por um corretor.

## Solução

- Responde perguntas usando dados do anuncio do imóvel
- Exemplos: "Quantas vagas de garagem?", "Qual o valor do condomínio?"
- Mantem o lead engajado enquanto coleta informacoes de qualificação

## Usuário

Leads que querem informacoes rapidas antes de decidir se tem interesse real.

## Valor

- Respostas instantaneas 24/7
- Reducao de perguntas repetitivas para corretores
- Melhor experiencia do lead com atendimento agil

## Dependências

Qualidade das informacoes depende dos dados cadastrados pela imobiliária nos anuncios.
