# Fast Lane para Leads Decididos

> Fluxo acelerado para leads que já estao decididos pelo imóvel do anuncio.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Leads que já sabem exatamente o que querem sao forçados a passar pelo mesmo fluxo extenso de qualificação, causando frustração e risco de desistencia.

## Solução

- Detecta quando lead indica que não quer ver outras opcoes
- Captura apenas informacoes essenciais: urgencia e interesse em visita
- Transfere rapidamente para corretor disponível
- Minimiza fricção no processo

## Usuário

Leads decididos que querem agilidade; corretores que valorizam leads quentes.

## Valor

- Reducao do tempo de qualificação para leads decididos
- Menor taxa de abandono desses leads
- Maior satisfacao do cliente com atendimento rapido

## Dependências

Triagem Inicial (para identificar perfil do lead).
