# Notificacao de Novo Lead via WhatsApp

> Corretores recebem notificacoes instantaneas quando um lead e atribuido a eles.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Corretores precisam verificar constantemente o CRM para saber se tem novos leads. Tempo de resposta lento reduz taxa de conversão.

## Solução

Notificacao via WhatsApp contendo:
- Dados da qualificação (urgencia, interesse em visita)
- Nome do lead e imóvel de interesse
- Origem do lead
- Botao para ver detalhes completos no CRM

## Usuário

Corretores que precisam responder rapidamente a novos leads.

## Valor

- Reducao do tempo de primeiro contato
- Corretor tem contexto antes de ligar para o lead
- Maior taxa de conversão por resposta rapida

## Dependências

Integracao Loft CRM (para distribuição e dados do lead).
