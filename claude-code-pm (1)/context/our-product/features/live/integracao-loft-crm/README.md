# Integracao com Loft/CRM

> Interface no CRM para gestao completa do processo de qualificação.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Dados de qualificação precisam estar disponiveis onde corretores já trabalham (no CRM). Sem integração, haveria duplicacao de ferramentas.

## Solução

Para administradores:
- Visualizacao de leads em qualificação e qualificados
- Lista com inicio, duracao, status, cliente
- Auditoria completa das conversas da IA

Para corretores:
- Acesso a todos os dados coletados
- Resumo estruturado (nome, urgencia, orcamento, preferencias)
- Historico da conversa com o lead

## Usuário

Administradores de imobiliárias e corretores usuários do Loft/CRM.

## Valor

- Dados de qualificação integrados ao fluxo de trabalho existente
- Visibilidade completa do processo de qualificação
- Auditoria e controle para gestores

## Dependências

Loft/CRM como plataforma base.
