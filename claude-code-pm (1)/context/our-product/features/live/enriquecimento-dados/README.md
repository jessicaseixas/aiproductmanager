# Enriquecimento de Dados

> Captura dados alem dos preenchidos no anuncio para criar um perfil mais completo do lead.

| | |
|---|---|
| **Status** | Produção |
| **Jira** | - |
| **Lançamento** | Out/2024 |

---

## Problema

Formulários de anuncios capturam apenas nome e telefone. Corretores perdem tempo fazendo perguntas basicas que poderiam ser coletadas automaticamente.

## Solução

Captura atraves de conversa natural:
- Urgencia para fechar negocio
- Interesse em visitar o imóvel
- Tipo de negocio (aluguel ou compra)
- Validacoes especificas (fiador para aluguel, financiamento para compra)
- Orcamento disponível
- Perfil do imóvel (localizacao, quartos, vagas, faixa de preco)

## Usuário

Corretores que precisam de leads pre-qualificados com informacoes completas.

## Valor

- Corretor recebe lead com perfil completo
- Reducao do tempo de primeiro contato
- Maior taxa de conversão por lead trabalhado

## Dependências

Nenhuma - feature core do produto.
