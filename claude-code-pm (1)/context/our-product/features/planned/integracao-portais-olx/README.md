# Integracao Portais OLX

> Permite receber leads dos portais do Grupo OLX (OLX, Viva Real, ZAP Imoveis).

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-418](https://loftbr.atlassian.net/browse/SLA-418) |
| **Lançamento** | Abr/2026 |

---

## Problema

Portais do Grupo OLX sao grandes fontes de leads no mercado imobiliario. Sem integração, imobiliárias precisam qualificar esses leads manualmente.

## Solução

- Integracao com APIs dos portais OLX, Viva Real e ZAP
- Recepcao automatica de leads submetidos nos portais
- Mapeamento de dados do lead para formato padrao
- Inicio automatico de qualificação via WhatsApp

## Usuário

Imobiliarias que anunciam nos portais do Grupo OLX.

## Valor

- Qualificacao automatica de leads de portais
- Maior volume de leads qualificados
- Tempo de resposta reduzido para leads de portais

## Dependências

Infraestrutura de qualificação existente.
