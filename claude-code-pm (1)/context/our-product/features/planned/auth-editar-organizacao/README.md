# Editar Organizacao

> Permite que administradores editem os dados da organização.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-524](https://loftbr.atlassian.net/browse/SLA-524) |
| **Lançamento** | Abr/2026 |

---

## Problema

Imobiliarias mudam de nome, logo, ou outras informacoes. O admin precisa manter os dados da organização atualizados.

## Solução

- Edicao de nome e dados da organização
- Upload de logo
- Configuracoes de branding para whitelabel
- Atualizacao de dados fiscais (CNPJ)

## Usuário

Administradores que precisam manter dados da organização atualizados.

## Valor

- Autonomia para manter dados corretos
- Personalizacao da experiencia whitelabel
- Dados fiscais atualizados para faturamento

## Dependências

Criar Organizacao (organização precisa existir).

Ver [TICKET.md](./TICKET.md) para especificações detalhadas.
