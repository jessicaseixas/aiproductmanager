# Listagem de Atendimentos

> Permite que usuários visualizem, pesquisem e filtrem todas as sessoes de qualificação de leads.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | - |
| **Lançamento** | Abr/2026 |

---

## Problema

Corretores e admins precisam de uma visao geral de todos os leads qualificados. Sem uma listagem central, não há como acompanhar o volume e status dos atendimentos.

## Solução

- Listagem páginada de todos os atendimentos
- Filtros por status, tipo de negocio, origem, data
- Busca por nome ou telefone do lead
- Ordenacao por diferentes critérios
- Acesso rapido aos detalhes de cada lead

## Usuário

Corretores que buscam leads para trabalhar; admins que monitoram operacao.

## Valor

- Visao centralizada de todos os leads
- Priorizacao baseada em filtros
- Acompanhamento de volume e status

## Dependências

Sign In (usuário precisa estar autenticado).
