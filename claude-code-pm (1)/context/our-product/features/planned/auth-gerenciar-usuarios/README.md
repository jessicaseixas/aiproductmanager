# Gerenciar Usuários

> Permite que administradores visualizem, editem e removam usuários da organização.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-523](https://loftbr.atlassian.net/browse/SLA-523) |
| **Lançamento** | Abr/2026 |

---

## Problema

Administradores precisam controlar quem tem acesso ao sistema. Corretores saem da imobiliária, mudam de funcao, ou precisam ter permissoes ajustadas.

## Solução

- Listagem de todos os usuários da organização
- Visualizacao de status (ativo, pendente, desativado)
- Edicao de dados e permissoes
- Desativacao/remocao de usuários

## Usuário

Administradores responsaveis pela gestao da equipe.

## Valor

- Controle de acesso centralizado
- Seguranca ao remover ex-funcionarios
- Visibilidade de quem tem acesso ao sistema

## Dependências

Convidar Membros (usuários precisam existir para serem gerenciados).
