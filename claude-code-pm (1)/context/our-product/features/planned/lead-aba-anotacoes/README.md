# Lead - Aba Anotacoes

> Permite que corretores criem, editem e excluam notas sobre o lead.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-513](https://loftbr.atlassian.net/browse/SLA-513) |
| **Lançamento** | Abr/2026 |

---

## Problema

Corretores precisam registrar observacoes proprias sobre o lead (ex: "ligou e pediu para retornar amanha", "prefere contato por WhatsApp").

## Solução

- Criação de notas com texto livre
- Edicao e exclusao de notas existentes
- Ordenacao cronologica das anotacoes
- Registro de autor e data de cada nota

## Usuário

Corretores que acompanham leads ao longo do tempo.

## Valor

- Registro de contexto adicional não capturado pela IA
- Continuidade no atendimento entre contatos
- Historico de interacoes humanas com o lead

## Dependências

Lead - Aba Detalhes (navegacao entre abas).
