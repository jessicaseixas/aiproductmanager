# Lead - Aba Detalhes

> Permite que corretores visualizem o resumo gerado por IA e informacoes de contato do lead.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-512](https://loftbr.atlassian.net/browse/SLA-512) |
| **Lançamento** | Abr/2026 |

---

## Problema

Corretores precisam acessar rapidamente informacoes essenciais do lead sem navegar por multiplas telas ou ler toda a conversa.

## Solução

- Exibe resumo estruturado gerado pela IA
- Mostra informacoes de contato (telefone, email)
- Apresenta dados de qualificação (urgencia, interesse em visita)
- Primeira aba da visao detalhada do lead

## Usuário

Corretores que precisam de acesso rapido ao perfil do lead no backoffice standalone.

## Valor

- Acesso instantaneo as informacoes mais importantes
- Contexto completo antes de contatar o lead
- Experiencia otimizada para dispositivos moveis

## Dependências

Listagem de Atendimentos (navegacao para detalhes do lead).

Ver [TICKET.md](./TICKET.md) para especificações detalhadas.
