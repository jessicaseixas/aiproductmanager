# Lead - Aba Interesse

> Exibe preferencias de imóvel e imóveis de interesse do lead.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-514](https://loftbr.atlassian.net/browse/SLA-514) |
| **Lançamento** | Abr/2026 |

---

## Problema

Corretor precisa saber exatamente o que o lead busca (tipo, localizacao, orcamento) e quais imóveis já demonstrou interesse.

## Solução

- Exibe preferencias coletadas (tipo de imóvel, localizacao, quartos, orcamento)
- Lista imóveis que o lead demonstrou interesse durante qualificação
- Permite entender o perfil de busca rapidamente

## Usuário

Corretores que precisam entender o perfil de busca do lead.

## Valor

- Corretores podem oferecer imóveis alinhados ao perfil
- Maior assertividade nas recomendacoes
- Reducao de tempo para entender o que o lead quer

## Dependências

Lead - Aba Detalhes (navegacao entre abas).

Ver [TICKET.md](./TICKET.md) para especificações detalhadas.
