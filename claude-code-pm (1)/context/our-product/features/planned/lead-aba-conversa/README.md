# Lead - Aba Conversa

> Exibe historico de mensagens trocadas entre o lead e o assistente de IA.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-511](https://loftbr.atlassian.net/browse/SLA-511) |
| **Lançamento** | Abr/2026 |

---

## Problema

Corretor pode precisar consultar o que foi conversado durante a qualificação para entender contexto, duvidas ou comentarios especificos do lead.

## Solução

- Exibe historico completo de mensagens
- Mostra mensagens do lead e respostas do assistente
- Permite scroll e busca na conversa
- Formato similar ao WhatsApp para familiaridade

## Usuário

Corretores que precisam revisar detalhes da conversa de qualificação.

## Valor

- Transparencia total sobre o que foi conversado
- Identificacao de duvidas não respondidas
- Auditoria do processo de qualificação

## Dependências

Lead - Aba Detalhes (navegacao entre abas).

Ver [TICKET.md](./TICKET.md) para especificações detalhadas.
