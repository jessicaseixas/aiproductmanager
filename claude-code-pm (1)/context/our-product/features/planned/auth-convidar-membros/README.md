# Convidar Membros

> Permite que administradores convidem outros administradores para a organização.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-522](https://loftbr.atlassian.net/browse/SLA-522) |
| **Lançamento** | Abr/2026 |

---

## Problema

Imobiliarias podem ter multiplos administradores. O admin precisa poder adicionar outros admins para gerenciar a organização.

## Solução

- Formulário para enviar convites por email
- Convite com link de ativacao e prazo de expiração
- Convidado entra como administrador (único perfil no MVP)
- Listagem de convites pendentes

## Usuário

Administradores que precisam adicionar outros administradores a organização.

## Valor

- Multiplos administradores para gerenciar a organização
- Onboarding self-service de novos admins
- Rastreabilidade de quem convidou quem

## Dependências

Criar Organizacao (precisa existir organização para convidar membros).
