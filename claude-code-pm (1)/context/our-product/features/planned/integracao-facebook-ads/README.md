# Integração Facebook Ads

> Permite receber leads gerados por campanhas de Facebook/Instagram Ads.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-419](https://loftbr.atlassian.net/browse/SLA-419) |
| **Lançamento** | Mar/2026 |

---

## Problema

Facebook e Instagram são canais importantes de geração de leads para imobiliárias. Sem integração, leads de campanhas precisam ser qualificados manualmente.

## Solução

- Integração com Facebook Lead Ads API
- Recepção automática de leads submetidos em formulários de anúncios
- Mapeamento de dados do lead para formato padrão
- Início automático de qualificação via WhatsApp

## Usuário

Imobiliárias que investem em campanhas de mídia paga no Facebook/Instagram.

## Valor

- Qualificação automática de leads de campanhas pagas
- Redução de tempo de resposta (lead qualificado em minutos)
- Melhor ROI de campanhas de mídia paga

## Dependências

Infraestrutura de qualificação existente. Implementação baseada na integração existente no Loft/CRM.
