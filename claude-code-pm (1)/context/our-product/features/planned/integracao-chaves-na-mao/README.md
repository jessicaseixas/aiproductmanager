# Integracao Chaves na Mao

> Permite receber leads do portal Chaves na Mao.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-455](https://loftbr.atlassian.net/browse/SLA-455) |
| **Lançamento** | Abr/2026 |

---

## Problema

Chaves na Mao e o terceiro portal com mais leads no ecossistema Loft (9% do volume). Sem integração, esses leads não sao qualificados automaticamente.

## Solução

- Integracao com API do Chaves na Mao
- Recepcao automatica de leads submetidos no portal
- Mapeamento de dados para formato padrao
- Inicio automatico de qualificação via WhatsApp

## Usuário

Imobiliarias que anunciam no Chaves na Mao.

## Valor

- Desbloqueio de segmento importante de leads
- Qualificacao automatica de mais uma fonte
- Expansao do alcance do produto standalone

## Dependências

Infraestrutura de qualificação existente.
