# Editar Meu Perfil

> Permite que usuários editem seus proprios dados de perfil.

| | |
|---|---|
| **Status** | Planejado |
| **Jira** | [SLA-528](https://loftbr.atlassian.net/browse/SLA-528) |
| **Lançamento** | Abr/2026 |

---

## Problema

Usuários precisam manter seus dados pessoais atualizados (nome, foto, telefone). Sem essa opcao, dependem do admin para qualquer alteracao.

## Solução

- Edicao de nome e foto de perfil
- Alteracao de email (com revalidação)
- Alteracao de senha
- Configuracoes de notificação

## Usuário

Todos os usuários do sistema (admins e corretores).

## Valor

- Autonomia do usuário sobre seus dados
- Reducao de solicitacoes ao admin
- Experiencia personalizada

## Dependências

Sign In (usuário precisa estar autenticado).

## Requisitos

Ver [TICKET.md](./TICKET.md) para especificações detalhadas.
